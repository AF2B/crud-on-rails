# Ruby API

# Technologies
- Ruby
- Ruby On Rails
- RSpec
- cpf_cnpj

- GET /users Lista todos os usuários.
- GET /users/:id Lista um usuário.
- POST /users Cria um usuário.
- PUT /users/:id Atualiza um usuário.
- DELETE /users/:id Deleta um usuário.
