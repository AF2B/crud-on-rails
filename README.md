# Ruby API

# Technologies
- Ruby
- Ruby On Rails
- RSpec
- cpf_cnpj

- GET /users Lista todos os usuários.
- GET /user/:id Lista um usuário.
- POST /user Cria um usuário.
- PUT /user/:id Atualiza um usuário.
- DELETE /user/:id Deleta um usuário.
